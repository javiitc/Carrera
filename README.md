# Carrera
## Como funciona?
En este programa se simula una carrera dónde tendrás 20 turnos para quedar primero. La carrera solo finalizará cuando pasen estos 20 turnos, tu objetivo será quedar primero o al menos entre los 3 mejores

Actuarás cómo el ingeniero a cargo de dar las instrucciones necesarias para que tu corredor gane la carrera, pero cuidado, porque según que decisión tomes, su nivel de estrés puede aumentar, y si se eleva mucho, corre riesgo de chocar y quedarse fuera de la carrera.

Buena suerte!
